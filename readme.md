# Tic Tac Toe Game

This is a simple **Tic Tac Toe game** built using **HTML, CSS, and JavaScript**.  
The game allows two players to play turn by turn and determines the winner or a draw based on standard Tic Tac Toe rules.

---

## Features

- Two-player gameplay (Player O and Player X)
- Winner detection based on predefined winning patterns
- Draw condition when all boxes are filled without a winner
- Reset and New Game functionality
- Interactive and responsive UI
- Clean and beginner-friendly JavaScript logic

---

## Technologies Used

- HTML
- CSS
- JavaScript (DOM Manipulation)

---

## How to Play

1. Player O starts the game.
2. Players take turns clicking on the empty boxes.
3. The game checks for a winner after every move.
4. If a player completes a winning pattern, the game displays the winner.
5. If all boxes are filled and no player wins, the game is declared a draw.
6. Use the **Reset** or **New Game** button to restart the game.

---

---

## Winning Logic

The game uses predefined winning patterns stored as index combinations.  
After every move, the game checks whether any pattern matches the current board state.
